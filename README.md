# Student Management System in Python

A **console-based Student Management System** built with Python and SQLite3.  
This project allows managing student records and their marks using a **menu-driven interface**.

---

## Features

- **Student Management**
  - Add new students
  - Update existing student details
  - Delete students
  - View all students

- **Marks Management**
  - Add marks for students
  - Update marks records
  - Delete marks records
  - View all marks along with student names

- **Database**
  - Uses **SQLite3** to store student and marks data
  - Enforces **foreign key constraints** to maintain data integrity

---

## Tech Stack

- Python 3.x  
- SQLite3 (built-in database)
